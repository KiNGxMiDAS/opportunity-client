## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

### Installation

1. 1. You will need an alchemy api key to run the client locally. Get a free Alchemy API Key at ([https://example.com](https://www.alchemy.com/)) You will need to create a .env file and add the variables from the .env_template.
2. Clone the repo
   ```sh
   git clone https://github.com/OpportunityProtocol/opportunity-client.git
   ```
3. Install NPM packages
   ```sh
   yarn install
   ```
4. Run the client locally
   ```sh
   yarn dev
   ```
